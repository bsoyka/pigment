# Pigment

**Pigment** is a set of Python utilities for colors.

```py
>>> red = Color(255, 0, 0)
>>> blue = Color(0, 0, 255)
>>> blend(red, blue).hsv
(300, 100, 50)
```

## Installation

Pigment is available on PyPI:

```console
$ python -m pip install pigment
```

Pigment officially supports Python 3.7+.
